📑 README.md
🚀 Guia de Distribuições & Kernels Linux
Este projeto é um guia interativo e visual desenvolvido para explorar o ecossistema Linux. O objetivo principal é consolidar e compartilhar o aprendizado sobre como o Kernel atua como o coração dos sistemas operacionais e as particularidades das principais distribuições (distros) do mercado.

O projeto foi construído do zero, focando em uma interface limpa, informativa e com uma pegada técnica voltada para entusiastas de hardware e segurança da informação.

🎯 Objetivo do Projeto
Diferente de sistemas complexos de gestão, este repositório nasceu da vontade de dividir o aprendizado. Ele serve como uma base de consulta rápida para entender:

O que é e o que faz o Kernel.

As diferenças entre distros como Ubuntu, Debian, Arch, Fedora, entre outras.

Pontos positivos e negativos de cada sistema para diferentes perfis de usuário (Devs, Gamers, SysAdmins).

🛠️ Tecnologias Utilizadas
HTML5: Estrutura semântica do conteúdo.

CSS3: Estilização personalizada com foco em legibilidade e design moderno (Dark Mode).

JavaScript: Implementação de interatividade, como efeitos de rolagem e dinamismo na página.

Git & GitHub: Versionamento e hospedagem do código.

📂 Estrutura de Arquivos
index.html: Onde reside toda a estrutura de texto e links das distribuições.

style.css: Responsável por toda a identidade visual, incluindo o card personalizado do Kernel.

script.js: Contém a lógica de interatividade da página.

/img: Pasta contendo os ativos visuais e diagramas explicativos.
